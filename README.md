# Banking-CLI-Application
A CLI based simple Banking application in Java and describing Banking core Functionalities like deposit, withdraw and able to see the transaction history.

---
        ===============================
               CLI BANKING SYSTEM      
        ===============================
        1. Deposit Funds
        2. Withdraw Funds
        3. View Transaction History
        4. View Account Details
        5. Exit
        -------------------------------
---

1. Deposit the amount in the account and able to see the account balance.
2. Withdraw the amount from the account.
3. Can able to see transaction history of deposit and withdrawal action with Local Date and Time.
4. Can able to see the Customer's account details.
5. To exit the CLI application.
